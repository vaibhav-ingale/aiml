Wildcard HTTPS Test

Wildcard HTTPS certificate is working!

Certificate Information

Domain: *.lab

Certificate: lab-cert.pem

Issuer: Vizen Lab CA

Supported Domains

This wildcard certificate works for:

  • *.lab - Any subdomain ending in .lab
  • lab - The root domain

Test Different Subdomains

Add these entries to your /etc/hosts file:

127.0.0.1 test.lab
127.0.0.1 api.lab
127.0.0.1 admin.lab
127.0.0.1 lab

Then test these URLs:

  • https://test.lab
  • https://api.lab
  • https://admin.lab
  • https://lab

Test Commands

# Test wildcard certificate
curl -k https://test.lab/
curl -k https://api.lab/health

# Test with CA certificate
curl --cacert ssl/ca/ca-cert.pem https://test.lab/
