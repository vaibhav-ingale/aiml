Spark Lab Services

Access all your Spark cluster services below

Spark Master

spark-master.lab

Main Spark cluster management interface

Spark Worker 1

spark-worker-1.lab

Worker node 1 monitoring interface

Spark Worker 2

spark-worker-2.lab

Worker node 2 monitoring interface

Spark Worker 3

spark-worker-3.lab

Worker node 3 monitoring interface

Spark History Server

spark-history.lab

Historical Spark application logs and metrics

Jupyter Lab

spark-jupyter.lab

Interactive notebook environment with Spark integration

MinIO API

minio-api.lab

S3-compatible object storage API endpoint

MinIO Console

minio-console.lab

Web-based MinIO administration interface

Setup Instructions

Add these entries to your /etc/hosts file:

127.0.0.1 spark-master.lab
127.0.0.1 spark-worker-1.lab
127.0.0.1 spark-worker-2.lab
127.0.0.1 spark-worker-3.lab
127.0.0.1 spark-history.lab
127.0.0.1 spark-jupyter.lab
127.0.0.1 minio-api.lab
127.0.0.1 minio-console.lab
127.0.0.1 spark-app.lab
